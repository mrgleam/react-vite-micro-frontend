# react-vite-micro-frontend

How to run the project

1. Clone the repository
2. Run `cd react-vite-micro-frontend`

Run micro-frontend

1. Run `cd todo-components`
2. Run `npm install`
3. Run `npm run build`
4. Run `npm run preview`

Run host-app

1. Run `cd host-app`
2. Run `npm install`
3. Run `npm run dev`
